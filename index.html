Habilidades

HTML

Tenho uma base sólida em HTML e domínio geral da linguagem, com um destaque especial para a semântica. Entendo a importância de estruturar corretamente os elementos para garantir acessibilidade, SEO e boas práticas de desenvolvimento. Sei como utilizar tags semânticas de forma eficiente para tornar o código mais organizado e compreensível, melhorando a experiência do usuário e a manutenção do projeto.

CSS

Tenho um bom domínio de CSS e sou capaz de estilizar páginas com eficiência. Minha maior qualidade está no uso do display: flex, sabendo aplicá-lo para criar layouts flexíveis e responsivos. Além disso, tenho habilidade em replicar layouts com grande precisão, garantindo que o design final fique o mais fiel possível ao modelo original, seja ele um wireframe ou uma referência visual.

JavaScript

Tenho um nível de conhecimento equivalente a um desenvolvedor júnior em JavaScript. Minha maior qualidade está na manipulação do DOM, onde me destaco ao modificar classes de elementos, manipular botões e criar elementos dinamicamente no documento. Além disso, sou capaz de trabalhar com eventos e interatividade para tornar as páginas mais dinâmicas e funcionais.

React

Estou no início da minha jornada com React, mas já consigo compreender e utilizar os principais hooks sem dificuldades. Tenho familiaridade com a estrutura de componentes, estado e efeitos, e estou constantemente aprimorando minha capacidade de criar interfaces dinâmicas e reutilizáveis.
